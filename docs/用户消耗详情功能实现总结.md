# 用户消耗详情功能实现总结

## 功能概述

在用户管理页面，管理员可以点击用户名，弹出用户的详细消耗分析，包括：
- 用户每日累计消耗的美金
- 各个套餐的每日消耗金额对比
- 不同模型的使用占比（柱状图和饼图）

## 实现架构

### 后端实现

#### 1. DTO 类型定义 (dto/analytics.go)

新增了以下数据结构：

```go
// UserConsumptionDetail - 主响应结构
type UserConsumptionDetail struct {
    UserInfo              UserBasicInfo              // 用户基础信息
    DailyConsumption      []DailyConsumptionItem     // 每日消耗明细
    PlanDailyConsumption  []PlanConsumptionDetail    // 套餐消耗详情
    ModelSummary          []ModelConsumptionSummary  // 模型总体统计
    Stats                 UserConsumptionStats       // 统计摘要
}

// DailyConsumptionItem - 每日消耗数据
type DailyConsumptionItem struct {
    Date         string                     // YYYY-MM-DD
    TotalUSD     float64                    // 当日总消耗（美金）
    RequestCount int                        // 请求次数
    Models       []ModelDailyConsumption    // 模型明细
}

// ModelDailyConsumption - 模型每日消耗
type ModelDailyConsumption struct {
    ModelName    string   // 模型名称
    USD          float64  // 消耗金额
    Quota        int      // 原始额度值
    RequestCount int      // 请求次数
    Percentage   float64  // 占当日总消耗的百分比
}

// PlanConsumptionDetail - 套餐消耗详情
type PlanConsumptionDetail struct {
    UserPlanID   int                // 用户套餐ID
    PlanName     string             // 套餐名称
    PlanType     string             // 套餐类型
    IsCurrent    int                // 是否为当前套餐
    DailyData    []PlanDailyData    // 每日数据
}

// ModelConsumptionSummary - 模型总体统计
type ModelConsumptionSummary struct {
    ModelName    string   // 模型名称
    TotalUSD     float64  // 总消耗
    RequestCount int      // 总请求数
    Percentage   float64  // 占总消耗的百分比
}

// UserConsumptionStats - 统计摘要
type UserConsumptionStats struct {
    TotalDays       int      // 统计天数
    TotalUSD        float64  // 总消耗
    AvgDailyUSD     float64  // 日均消耗
    PeakDailyUSD    float64  // 峰值消耗
    TotalRequests   int      // 总请求数
}
```

#### 2. Service 层 (service/user_consumption_service.go)

核心函数：

- **GetUserConsumptionDetail(userId, days)** - 主查询函数，聚合所有数据
- **getUserDailyConsumption()** - 查询每日消耗并按模型分组
- **getUserPlanConsumption()** - 查询套餐维度的消耗
- **getUserModelSummary()** - 查询模型总体统计
- **getDayModelBreakdown()** - 获取特定日期的模型明细
- **calculateConsumptionStats()** - 计算统计摘要

数据来源：
- 从 `logs` 表聚合数据（type=2 为消耗日志）
- 按日期、模型、套餐分组统计
- 自动转换 quota 为 USD

## 使用说明

### 管理员操作

1. 进入用户管理页面
2. 点击任意用户的用户名（蓝色下划线）
3. 弹出用户消耗详情侧边栏
4. 可以：
   - 查看用户基础信息和统计概览
   - 切换 Tab 查看不同维度的图表
   - 修改天数范围（7/15/30/60/90天）
   - 查看模型使用明细表格

### 数据说明

- **消耗金额**：所有金额单位为 USD（美元）
- **时间范围**：默认30天，最多90天
- **数据来源**：从消耗日志（logs 表 type=2）聚合
- **实时性**：每次打开弹窗重新请求最新数据
- **时区**：使用北京时间（UTC+8）
