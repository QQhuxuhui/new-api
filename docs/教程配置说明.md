# 教程系统配置说明

## 问题原因

您创建的 HTML 教程文件（`docs/tutorial-*.html`）**不会自动显示**，因为系统使用的是**数据库驱动的动态教程管理**。

## 系统架构

### 后端（Go）
- **配置存储**：教程内容存储在数据库的 `console_setting` 配置中
- **字段说明**：
  - `tutorial`：教程 JSON 字符串（包含所有章节）
  - `tutorial_enabled`：是否启用教程功能（默认 true）

### 前端（React）
- **路由地址**：`/tutorial`
- **显示条件**：
  1. `tutorial_enabled === true` （后端配置）
  2. `HeaderNavModules.tutorial === true` （顶栏模块配置）
  3. 教程内容不为空

## 配置教程的步骤

### 方式一：通过管理后台配置（推荐）

1. **登录管理后台**（需要管理员权限）

2. **进入设置** → **仪表盘设置** → **教程管理**

3. **启用教程功能**：确保"启用教程面板"开关已打开

4. **添加教程章节**：

#### Claude Code 教程
```json
{
  "id": "claude-code-npm",
  "title": "Claude Code 安装与配置（npm 方式）",
  "order": 1,
  "enabled": true,
  "format": "html",
  "content": "将 docs/tutorial-claude-code.html 的完整内容复制到这里"
}
```

#### Codex CLI 教程
```json
{
  "id": "codex-cli-setup",
  "title": "OpenAI Codex CLI 安装与配置",
  "order": 2,
  "enabled": true,
  "format": "html",
  "content": "将 docs/tutorial-codex.html 的完整内容复制到这里"
}
```

#### Gemini CLI 教程
```json
{
  "id": "gemini-cli-setup",
  "title": "Google Gemini CLI 安装与配置",
  "order": 3,
  "enabled": true,
  "format": "html",
  "content": "将 docs/tutorial-gemini-cli.html 的完整内容复制到这里"
}
```

5. **保存配置**

6. **访问教程页面**：
   - 顶部导航栏会出现"教程"入口
   - 点击进入 `/tutorial` 页面查看

### 方式二：通过 API 配置

如果您有数据库访问权限，可以直接更新配置：

1. 找到 `options` 表中的 `console_setting` 配置项
2. 更新 JSON 配置的 `tutorial` 和 `tutorial_enabled` 字段

## 教程JSON格式说明

教程配置使用以下 JSON 格式：

```json
{
  "sections": [
    {
      "id": "unique-id",           // 唯一标识符（小写字母、数字、连字符）
      "title": "教程标题",          // 显示标题
      "order": 1,                  // 排序顺序（数字）
      "enabled": true,             // 是否启用
      "format": "html",            // 格式：html 或 markdown
      "content": "<div>...</div>"  // 教程内容
    }
  ]
}
```

### 字段验证规则

- **id**：
  - 必填
  - 只能包含小写字母、数字和连字符
  - 长度不超过 50 字符
  - 不能重复

- **title**：
  - 必填
  - 长度不超过 100 字符

- **order**：
  - 必填
  - 非负整数
  - 决定显示顺序

- **format**：
  - 必填
  - 只能是 `html` 或 `markdown`

- **content**：
  - 必填
  - 长度不超过 50000 字符

## 访问教程页面

配置完成后，教程可以通过以下方式访问：

1. **导航栏入口**：
   - 顶部导航栏会显示"教程"链接
   - 前提：`tutorial_enabled === true` 且 `HeaderNavModules.tutorial === true`

2. **直接访问**：
   - URL：`https://your-domain.com/tutorial`

## 故障排查

### 教程入口不显示

1. **检查教程启用状态**：
   - 管理后台 → 仪表盘设置 → 教程管理
   - 确认"启用教程面板"已开启

2. **检查顶栏模块配置**：
   - 管理后台 → 设置 → 系统设置 → 模块管理
   - 查看 `HeaderNavModules` 配置
   - 确保 `tutorial: true`

3. **检查浏览器控制台**：
   - 打开浏览器开发者工具
   - 查看 `/api/status` 接口返回的数据
   - 确认 `tutorial_enabled` 和 `tutorial` 字段存在

### 教程内容不显示

1. **检查 JSON 格式**：
   - 确保教程配置是有效的 JSON
   - 使用 JSON 验证工具检查格式

2. **检查内容长度**：
   - 每个章节内容不超过 50000 字符
   - 总章节数不超过 20 个

3. **检查章节启用状态**：
   - 确保章节的 `enabled: true`
