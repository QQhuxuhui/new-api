🏄 Windsurf IDE 配置教程

步骤 1: 安装 Windsurf

前往 windsurf.com 下载 Windows 安装包（支持 x64 和 ARM64）

💡 系统要求：Windows 10/11，推荐 16GB 内存

前往 windsurf.com 下载 macOS 安装包

💡 系统要求：macOS 11 (Big Sur) 或更高版本，推荐 16GB 内存

前往 windsurf.com 下载 Linux 安装包

💡 推荐 16GB 内存

步骤 2: 配置中转 API（环境变量方式）

在 PowerShell 配置文件中添加：

$env:OPENAI_API_BASE="{{OPENAI_API_URL}}"
$env:OPENAI_API_KEY="YOUR_API_KEY"

配置后重启 Windsurf 生效

编辑 ~/.zshrc 或 ~/.bashrc：

export OPENAI_API_BASE="{{OPENAI_API_URL}}"
export OPENAI_API_KEY="YOUR_API_KEY"

# 重新加载配置
source ~/.zshrc

配置后从终端启动 Windsurf 以使环境变量生效

编辑 ~/.bashrc 或 ~/.zshrc：

export OPENAI_API_BASE="{{OPENAI_API_URL}}"
export OPENAI_API_KEY="YOUR_API_KEY"

# 重新加载配置
source ~/.bashrc

配置后从终端启动 Windsurf 以使环境变量生效

📝 配置说明：

  • API Base URL: {{OPENAI_API_URL}}
  • Windsurf 使用 OpenAI 兼容格式的 API
  • 环境变量对所有项目全局生效
  • 修改配置后需要重启 Windsurf

步骤 3: 验证配置

打开 Windsurf，尝试使用 AI 功能：

  • 使用 AI Chat 功能测试连接
  • 检查是否能正常获得 AI 响应
