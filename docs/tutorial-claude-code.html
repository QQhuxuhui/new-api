🤖 Claude Code 安装与配置

步骤 1: 安装 Claude Code

Windows 安装方式：

打开 PowerShell，执行以下命令：

irm https://claude.ai/install.ps1 | iex

💡 提示：安装完成后重启终端

macOS 安装方式（二选一）：

方式 1：使用官方脚本

curl -fsSL https://claude.ai/install.sh | bash

方式 2：使用 Homebrew（推荐）

brew install --cask claude-code

Linux / WSL2 安装方式：

curl -fsSL https://claude.ai/install.sh | bash

💡 安装后可能需要重启终端或执行 source ~/.bashrc

验证安装：

claude --version

显示版本号即表示安装成功

步骤 2: 配置中转 API

推荐方式：全局配置文件

创建配置文件：%USERPROFILE%\.claude\settings.json

使用以下内容：

{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}

创建配置目录和文件：

mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}
EOF

创建配置目录和文件：

mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}
EOF

📝 配置说明：

  • API Base URL: {{CLAUDE_API_URL}}
  • API Key: 从您的账户获取（通常以 sk- 开头）
  • 配置文件存储在用户目录，安全且所有项目共享

步骤 3: 开始使用

进入项目目录，运行：

cd your-project
claude

现在可以通过自然语言与 Claude Code 交互了！
