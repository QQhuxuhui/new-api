🤖 Claude Code 安装与配置

前置要求

系统要求：

  • Node.js 18.0.0 或更高版本
  • npm 9.0.0 或更高版本
  • 支持 Windows 10+、macOS 10.15+、Ubuntu 20.04+/Debian 10+

检查 Node.js 和 npm 版本：

node --version
npm --version

💡 如果版本不符合要求，请先从 nodejs.org 下载安装最新版本

步骤 1: 安装/更新 Node.js（如果需要）

Windows 安装方式：

方式 1：使用官方安装包（推荐）

  1. 访问 nodejs.org
  2. 下载 LTS 版本（长期支持版）
  3. 运行安装程序，按默认选项安装
  4. 安装完成后重启终端

方式 2：使用 Chocolatey（如果已安装）

choco install nodejs

macOS 安装方式：

方式 1：使用 Homebrew（推荐）

brew install node

方式 2：使用官方安装包

  1. 访问 nodejs.org
  2. 下载 macOS 安装包
  3. 运行 .pkg 文件完成安装

方式 3：使用 nvm（Node 版本管理器）

# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node.js LTS
nvm install --lts
nvm use --lts

Linux 安装方式：

方式 1：使用 NodeSource 仓库（推荐）

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
npm --version

方式 2：使用 nvm（Node 版本管理器）

# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新加载配置
source ~/.bashrc  # 或 source ~/.zshrc

# 安装 Node.js LTS
nvm install --lts
nvm use --lts

步骤 2: 使用 npm 安装 Claude Code

全局安装 Claude Code：

Windows 安装：

npm install -g @anthropic-ai/claude-code

💡 如遇权限问题，请使用管理员权限运行 PowerShell

macOS 安装：

npm install -g @anthropic-ai/claude-code

如果遇到权限错误，使用 sudo：

sudo npm install -g @anthropic-ai/claude-code

💡 推荐配置 npm 使用用户目录，避免使用 sudo

Linux 安装：

npm install -g @anthropic-ai/claude-code

如果遇到权限错误，使用 sudo：

sudo npm install -g @anthropic-ai/claude-code

验证安装：

claude --version

显示版本号即表示安装成功（例如：claude version 1.0.58）

🔧 常见安装问题：

  • 命令未找到：重启终端或检查 npm 全局路径配置
  • 权限错误：Windows 使用管理员权限；macOS/Linux 使用 sudo
  • 网络超时：配置 npm 镜像源（如使用淘宝镜像）

步骤 3: 配置 npm 镜像（可选）

使用淘宝镜像加速下载（中国用户推荐）：

# 临时使用
npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com

# 永久配置
npm config set registry https://registry.npmmirror.com

# 恢复官方源
npm config set registry https://registry.npmjs.org

步骤 4: 配置中转 API

推荐方式：全局配置文件

Windows 配置：

1. 创建配置目录和文件：

# 在 PowerShell 中执行
$configDir = "$env:USERPROFILE\.claude"
if (!(Test-Path $configDir)) {
    New-Item -ItemType Directory -Path $configDir
}

# 创建配置文件
@"
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}
"@ | Out-File -FilePath "$configDir\settings.json" -Encoding utf8

2. 或手动创建文件：

  • 位置：%USERPROFILE%\.claude\settings.json
  • 例如：C:\Users\YourName\.claude\settings.json

macOS 配置：

创建配置目录和文件：

mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}
EOF

Linux 配置：

创建配置目录和文件：

mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_API_KEY": "YOUR_API_KEY"
  }
}
EOF

📝 配置说明：

  • API Base URL: {{CLAUDE_API_URL}}
  • API Key: 将 YOUR_API_KEY 替换为您的实际 API Key（通常以 sk- 开头）
  • 配置位置: 存储在用户目录下，安全且所有项目共享
  • 配置格式: 必须是有效的 JSON 格式

步骤 5: 验证配置

运行诊断命令：

claude doctor

该命令会检查：

  • Claude Code 版本和安装类型
  • 配置文件是否正确
  • API 连接是否正常

💡 如果诊断显示问题，请检查配置文件的 JSON 格式和 API Key 是否正确

步骤 6: 开始使用

基本使用：

# 进入项目目录
cd your-project

# 启动 Claude Code
claude

常用命令：

# 查看帮助
claude --help

# 查看版本
claude --version

# 运行诊断
claude doctor

# 更新到最新版本
npm update -g @anthropic-ai/claude-code

💡 现在可以通过自然语言与 Claude Code 交互，进行代码生成、调试、重构等任务了！

更新与卸载

更新 Claude Code：

npm update -g @anthropic-ai/claude-code

卸载 Claude Code：

npm uninstall -g @anthropic-ai/claude-code

清理配置文件（可选）：

# Windows (PowerShell)
Remove-Item -Recurse -Force "$env:USERPROFILE\.claude"
# macOS/Linux
rm -rf ~/.claude
# Linux
rm -rf ~/.claude
