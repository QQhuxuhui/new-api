🤖 Claude Code 安装与配置

前置要求

系统要求：

  • Node.js 18.0.0 或更高版本
  • npm 9.0.0 或更高版本
  • 支持 Windows 10+、macOS 10.15+、Ubuntu 20.04+/Debian 10+

检查 Node.js 和 npm 版本：

node --version
npm --version

💡 如果版本不符合要求，请先从 nodejs.org 下载安装最新 LTS 版本

步骤 1: 安装 Node.js（如果需要）

Windows：

访问 nodejs.org 下载 LTS 版本安装包，运行安装程序，按默认选项安装，安装完成后重启终端。

macOS：

使用 Homebrew 安装（推荐）：

brew install node

Linux：

使用 NodeSource 仓库安装：

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
npm --version

步骤 2: 使用 npm 安装 Claude Code

全局安装 Claude Code：

# Windows/macOS/Linux 通用命令
npm install -g @anthropic-ai/claude-code

💡 权限问题解决：
• Windows：使用管理员权限运行 PowerShell
• macOS/Linux：如遇权限错误，使用 sudo npm install -g @anthropic-ai/claude-code

验证安装：

claude --version

显示版本号即表示安装成功（例如：claude version 1.0.58）

🔧 常见问题：

  • 命令未找到：重启终端或检查 npm 全局路径配置
  • 网络超时：中国用户可配置淘宝镜像：npm config set registry https://registry.npmmirror.com

步骤 3: 配置中转 API

创建配置文件（推荐方式）：

Windows：

在 PowerShell 中执行：

$configDir = "$env:USERPROFILE\.claude"
if (!(Test-Path $configDir)) {
    New-Item -ItemType Directory -Path $configDir
}

@"
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
  }
}
"@ | Out-File -FilePath "$configDir\settings.json" -Encoding utf8

macOS / Linux：

在终端中执行：

mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
  }
}
EOF

📝 配置说明：

  • API Base URL: {{CLAUDE_API_URL}}
  • API Key: 将 YOUR_API_KEY 替换为您的实际 API Key（通常以 sk- 开头）
  • 注意: 配置文件中使用 ANTHROPIC_AUTH_TOKEN（不是 ANTHROPIC_API_KEY）
  • 配置位置:
    • Windows: %USERPROFILE%\.claude\settings.json
    • macOS/Linux: ~/.claude/settings.json

步骤 4: 验证配置

运行诊断命令：

claude doctor

该命令会检查：

  • Claude Code 版本和安装类型
  • 配置文件是否正确
  • API 连接是否正常

💡 如果诊断显示问题，请检查配置文件的 JSON 格式和 API Key 是否正确

步骤 5: 开始使用

启动 Claude Code：

# 进入项目目录
cd your-project

# 启动 Claude Code
claude

常用命令：

# 查看帮助
claude --help

# 查看版本
claude --version

# 运行诊断
claude doctor

# 更新到最新版本
npm update -g @anthropic-ai/claude-code

💡 现在可以通过自然语言与 Claude Code 交互，进行代码生成、调试、重构等任务了！
