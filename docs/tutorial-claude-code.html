Claude Code 安装与配置

高效的 AI 编程助手配置指南

0

前置要求

系统要求：

  • Node.js 18.0.0 或更高版本
  • npm 9.0.0 或更高版本
  • 支持 Windows 10+、macOS 10.15+、Ubuntu 20.04+/Debian 10+

检查版本：

node --version
npm --version
💡 提示： 如果版本不符合要求，请先从 nodejs.org 下载安装最新 LTS 版本。
1

安装 Node.js

🪟 Windows

访问 nodejs.org 下载 LTS 版本安装包，按默认选项安装并重启终端。

🍎 macOS

推荐使用 Homebrew 安装：

brew install node
🐧 Linux

使用 NodeSource 仓库安装：

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
npm --version
2

安装 Claude Code

全局安装命令：

# Windows/macOS/Linux 通用
npm install -g @anthropic-ai/claude-code
⚠️ 权限提示：
  • Windows： 请使用管理员权限运行 PowerShell
  • macOS/Linux： 如遇报错，请尝试 sudo npm install -g ...

验证安装：

claude --version

成功示例：claude version 1.0.58

3

配置中转 API

配置中转 API 以使用 Claude Code。请选择您的操作系统：

🪟 Windows (PowerShell)
$configDir = "$env:USERPROFILE\.claude"
if (!(Test-Path $configDir)) {
    New-Item -ItemType Directory -Path $configDir
}

@"
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
  }
}
"@ | Out-File -FilePath "$configDir\settings.json" -Encoding utf8
🍎 macOS / 🐧 Linux
mkdir -p ~/.claude
cat > ~/.claude/settings.json << 'EOF'
{
  "env": {
    "ANTHROPIC_BASE_URL": "{{CLAUDE_API_URL}}",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
  }
}
EOF
📝 配置要点：
  • API Base URL: {{CLAUDE_API_URL}}
  • API Key: 将 YOUR_API_KEY 替换为您的实际密钥 (sk-...)
  • 注意: 配置文件中使用 ANTHROPIC_AUTH_TOKEN 字段
4

验证与使用

1. 运行诊断：

claude doctor

此命令将检查版本、配置文件及 API 连接状态。

2. 启动工具：

cd your-project
claude
claude --help 查看帮助
claude --version 查看版本
npm update -g ... 更新版本
