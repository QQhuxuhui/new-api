✨ Google Gemini CLI 安装与配置

前置要求

系统要求：

  • Node.js 18+ 或更高版本（推荐 Node 20+）
  • 支持 Windows、macOS 和 Linux

检查 Node.js 版本：

node --version

💡 如果版本低于 18，请先升级 Node.js

步骤 1: 安装 Gemini CLI

使用 npm 全局安装（所有平台通用）：

npm install -g @google/gemini-cli

Windows 用户注意事项：

  • 建议使用管理员权限运行 PowerShell 或 CMD
  • 如遇权限问题，可能需要配置 npm 全局安装路径

macOS/Linux 用户注意事项：

  • 如遇权限问题，可能需要使用 sudo：sudo npm install -g @google/gemini-cli
  • 或配置 npm 使用用户级全局目录（推荐）

验证安装：

gemini --version

显示版本号即表示安装成功

步骤 2: 配置 API Key

获取 Gemini API Key：

  1. 访问 Google AI Studio
  2. 登录您的 Google 账户
  3. 在控制台中创建新的 API Key

配置 API Key（三种方式）：

方式 1：首次运行时交互式配置（推荐）

gemini

首次运行时会提示输入 API Key，按照提示完成配置即可

方式 2：使用环境变量

Windows (PowerShell)：

$env:GEMINI_API_KEY="YOUR_API_KEY"

Windows (CMD)：

set GEMINI_API_KEY=YOUR_API_KEY

macOS/Linux (编辑 ~/.zshrc 或 ~/.bashrc)：

export GEMINI_API_KEY="YOUR_API_KEY"

# 重新加载配置
source ~/.zshrc  # 或 source ~/.bashrc

方式 3：使用配置文件

创建配置文件（路径因系统而异）：

  • Windows: %APPDATA%\gemini-cli\config.json
  • macOS/Linux: ~/.config/gemini-cli/config.json
{
  "apiKey": "YOUR_API_KEY",
  "model": "gemini-pro"
}

📝 中转 API 配置（如果使用）：

如果您使用的是中转 API 服务，可以在配置文件中添加：

{
  "apiKey": "YOUR_API_KEY",
  "baseUrl": "{{GEMINI_API_URL}}",
  "model": "gemini-pro"
}
  • API Base URL: {{GEMINI_API_URL}}
  • 确保中转服务兼容 Gemini API 格式

步骤 3: 开始使用

基本使用：

# 启动交互式对话
gemini

# 直接提问
gemini "What is the capital of France?"

# 使用管道输入
echo "Explain quantum computing" | gemini

常用功能：

  • 代码生成和解释
  • 文本翻译和摘要
  • 问答和知识查询
  • 创意写作辅助

💡 Gemini CLI 支持多种内置工具，可用于文件处理、代码分析等任务

高级配置选项

配置文件完整示例：

{
  "apiKey": "YOUR_API_KEY",
  "model": "gemini-pro",
  "temperature": 0.7,
  "maxTokens": 2048,
  "topP": 0.9
}

支持的模型：

  • gemini-pro - 通用模型（默认）
  • gemini-pro-vision - 支持图像输入
  • gemini-ultra - 最强大的模型（需相应权限）
