Google Gemini CLI 安装与配置

Google AI 官方命令行工具使用指南

0

前置要求

系统要求：

  • Node.js 18+ (推荐 Node 20+)
  • 支持 Windows、macOS 和 Linux

检查环境：

node --version
💡 提示： 如果版本低于 18，请先升级 Node.js。
1

安装 Node.js

🪟 Windows

访问 nodejs.org 下载 LTS 版本安装包，按默认选项安装并重启终端。

🍎 macOS

推荐使用 Homebrew 安装：

brew install node
🐧 Linux

使用 NodeSource 仓库安装：

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
2

安装 Gemini CLI

全局安装命令：

npm install -g @google/gemini-cli
⚠️ 权限提示：
  • Windows： 请使用管理员权限运行 PowerShell
  • macOS/Linux： 如遇报错，请尝试 sudo npm install -g ...

验证安装：

gemini --version
3

配置中转 API

配置环境变量：

🪟 Windows (PowerShell)
# 临时设置
$env:GEMINI_API_KEY="YOUR_API_KEY"
$env:GOOGLE_GEMINI_BASE_URL="{{BASE_URL}}"

# 永久设置
[System.Environment]::SetEnvironmentVariable('GEMINI_API_KEY', 'YOUR_API_KEY', 'User')
[System.Environment]::SetEnvironmentVariable('GOOGLE_GEMINI_BASE_URL', '{{BASE_URL}}', 'User')
🍎 macOS / 🐧 Linux
# 编辑 ~/.bashrc 或 ~/.zshrc
export GEMINI_API_KEY="YOUR_API_KEY"
export GOOGLE_GEMINI_BASE_URL="{{BASE_URL}}"

# 重新加载
source ~/.zshrc
📝 配置要点：
  • API Base URL： {{BASE_URL}} 替换为您的中转服务地址
  • API Key： 将 YOUR_API_KEY 替换为您的实际密钥
  • 注意： 必须同时设置 GEMINI_API_KEY 和 GOOGLE_GEMINI_BASE_URL 两个环境变量
4

开始使用

交互模式：

gemini

单次提问：

gemini "Explain quantum computing briefly"

管道输入：

cat file.txt | gemini "Summarize this file"
代码生成 编写与解释代码
文本处理 翻译、摘要、润色
知识问答 百科全书式查询
