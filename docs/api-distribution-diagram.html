1
客户端请求
下游客户端使用分发的令牌向 API 代理服务发起 HTTPS 请求
2
令牌验证
代理服务验证令牌有效性，检查配额和权限
3
请求转发
替换为上游 API Key，将请求转发至上游服务器
4
响应返回
上游响应经代理返回给客户端，记录用量
