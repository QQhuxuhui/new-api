💻 Cursor IDE 配置教程

步骤 1: 安装 Cursor

前往 cursor.sh 下载 Windows 安装包并安装

前往 cursor.sh 下载 macOS 安装包并安装

前往 cursor.sh 下载 Linux 安装包并安装

步骤 2: 配置中转 API

推荐方式：在 Cursor 设置中配置

  1. 打开 Cursor，进入设置
    • Windows/Linux: Ctrl + ,
    • macOS: Cmd + ,
  2. 搜索 "Override OpenAI Base URL"
  3. 启用该选项并填写： 
    Base URL: {{OPENAI_API_URL}}
  4. 在 API Key 设置中填入您的 API Key

📝 配置说明：

  • API Base URL: {{OPENAI_API_URL}}
  • 注意：Cursor 需要 OpenAI 格式的端点（包含 /v1 后缀）
  • 配置存储在 Cursor 内部，不会暴露到项目文件

替代方式: 使用环境变量

在 PowerShell 配置文件中添加：

$env:OPENAI_API_BASE="{{OPENAI_API_URL}}"
$env:OPENAI_API_KEY="YOUR_API_KEY"

编辑 ~/.zshrc 或 ~/.bashrc，添加：

export OPENAI_API_BASE="{{OPENAI_API_URL}}"
export OPENAI_API_KEY="YOUR_API_KEY"

# 重新加载配置
source ~/.zshrc

编辑 ~/.bashrc，添加：

export OPENAI_API_BASE="{{OPENAI_API_URL}}"
export OPENAI_API_KEY="YOUR_API_KEY"

# 重新加载配置
source ~/.bashrc
