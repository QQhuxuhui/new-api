🤖 OpenAI Codex CLI 安装与配置

步骤 1: 安装 Codex CLI

Windows 安装方式（推荐使用 WSL）：

💡 Windows 支持仍在实验阶段，建议在 WSL 中运行

方式 1：使用 npm（全局安装）

npm install -g @openai/codex

方式 2：下载预编译二进制文件

从 GitHub Releases 下载最新的 Windows 版本

macOS 安装方式（三选一）：

方式 1：使用 Homebrew（推荐）

brew install --cask codex

方式 2：使用 npm（全局安装）

npm install -g @openai/codex

方式 3：下载预编译二进制文件

从 GitHub Releases 下载 macOS 版本（如 codex-aarch64-apple-darwin）

Linux 安装方式（二选一）：

方式 1：使用 npm（全局安装）

npm install -g @openai/codex

方式 2：下载预编译二进制文件

从 GitHub Releases 下载 Linux 版本（如 codex-x86_64-unknown-linux-musl）

# 下载后解压并添加到 PATH
chmod +x codex
sudo mv codex /usr/local/bin/

系统要求：

  • macOS 或 Linux（Windows 需使用 WSL）
  • Node.js（如果使用 npm 安装）

验证安装：

codex --version

显示版本号即表示安装成功

步骤 2: 配置认证

首次运行时会自动引导认证：

codex

认证方式（二选一）：

方式 1：使用 ChatGPT 账户登录（推荐）

  • 支持账户类型：ChatGPT Plus、Pro、Business、Edu 或 Enterprise
  • 在提示时选择 "Sign in with ChatGPT"
  • 按照浏览器提示完成 OAuth 认证

方式 2：使用 API Key（高级控制）

创建配置文件：~/.codex/config.toml

[api]
key = "YOUR_OPENAI_API_KEY"
base_url = "{{OPENAI_API_URL}}"  # 使用中转API时配置

📝 中转 API 配置：

  • API Base URL: {{OPENAI_API_URL}}
  • 将您的 API Key 替换到配置文件中
  • 配置文件位置：~/.codex/config.toml
  • 支持丰富的配置选项，详见 官方文档

步骤 3: 开始使用

直接在终端运行：

codex

或使用自然语言命令：

codex "list all files in current directory"

💡 Codex CLI 可以将自然语言转换为对应的 Bash/ZShell/PowerShell 命令
