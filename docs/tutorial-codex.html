OpenAI Codex CLI 安装与配置

OpenAI 官方命令行工具使用指南

0

前置要求

支持平台：

  • macOS 10.15+
  • Linux (Ubuntu, Debian, Fedora 等)
  • Windows (仅通过 WSL 支持)

环境依赖：

node --version
💡 提示： Codex CLI 依赖 Node.js 环境，请确保已安装最新 LTS 版本。
1

安装 Node.js

🪟 Windows

请先安装 WSL (Windows Subsystem for Linux)，然后在 WSL 环境中按照 Linux 步骤操作。

🍎 macOS

推荐使用 Homebrew 安装：

brew install node
🐧 Linux

使用 NodeSource 仓库安装：

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
2

安装 Codex CLI

根据平台选择安装方式：

🍎 macOS (Homebrew 推荐)
brew install --cask codex
🐧 Linux / WSL (npm)
npm install -g @openai/codex

验证安装：

codex --version
3

配置中转 API

1. 设置环境变量：

export OPENAI_API_KEY="YOUR_API_KEY"
echo $OPENAI_API_KEY | codex login --with-api-key

2. 配置中转接口：

编辑配置文件 ~/.codex/config.toml：

preferred_auth_method = "apikey"

[model_providers]
[model_providers.openai]
base_url = "{{OPENAI_API_URL}}"
📝 配置要点：
  • API Base URL： {{OPENAI_API_URL}} 替换为您的中转服务地址
  • API Key： 将 YOUR_API_KEY 替换为您的实际密钥 (sk-...)
4

使用指南

基本命令模式：

codex "list all files in current directory"

Codex 会将您的自然语言描述转换为对应的 Shell 命令。

常用功能：

  • 💡 自然语言转命令： 描述你想做的事情，Codex 生成命令。
  • 🔄 解释命令： 加上 --explain 参数让 Codex 解释命令含义。
  • 🛠️ 脚本生成： 可以生成复杂的 Shell 或 Python 脚本。
