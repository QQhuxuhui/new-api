2.1 **"平台"或"本平台"**：指 Spark Code 大模型网关与AI资产管理系统，包括其网站、API接口及相关服务。

### 第六条 服务说明

6.1 **服务性质**：本平台提供的是 AI 模型 API 网关服务，即通过统一接口将用户请求转发至各上游 AI 服务提供商。**本平台不直接提供 AI 模型或生成 AI 内容，不对上游服务的输出内容承担责任**。

6.2 **服务范围**：包括但不限于：
- 兼容 OpenAI API 格式的接口服务
- Claude、Gemini 等多种 AI 模型的接口转换
- 图像生成（如 Midjourney）、音乐生成（如 Suno）等接口
- 嵌入（Embeddings）、重排序（Rerank）等辅助功能
- 实时对话（Realtime API）等高级功能

6.3 **服务可用性**：我们将尽合理努力保持服务稳定运行，但**不保证服务不间断、无错误或完全安全**。服务可能因以下原因中断：
- 系统维护或升级
- 网络故障
- 上游服务不可用
- 不可抗力事件
- 其他超出平台控制范围的因素

7.1 **合法使用**：您承诺仅将本服务用于合法目的，遵守中华人民共和国法律法规及您所在地区的适用法律。

12.1 **退款情形**：以下情况可申请退款：
